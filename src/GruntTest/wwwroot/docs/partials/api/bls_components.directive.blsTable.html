blsTable
directive in module bls_components

Description

blsTable directive Note: Id must be specified

Dependencies

  • $log
  • $timeout

Usage

as element:
<bls-table>
</bls-table>

Directive info

  • This directive creates new scope.

Properties

  • data

    The value in the model that the control is bound to.

  • $modelValue

    The value in the model that the control is bound to.

    data: '=ngModel', funcAsync: '&', getChildren: '&', options: '=', totalItems: '=', childItemsProp: '@'

Example

<bls-table ng-model="model.data"
     func-async="query(pageIndex, pageLength, searchedText, sortTable, filters)"
     options="options"
     total-items="model.totalItems"
     id="btbSampleExample">
<bls-cols>
 <bls-col resize dragable sort field-name="id"></bls-col>

     <a title="company!" href="javascript: void(0)" ng-click="options.callbacks[0](::row)">
         {{::row.company}}
     </a>

 </bls-col>
